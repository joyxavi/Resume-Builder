# 🧠 Smart Resume Builder – Canva Inspired

Build stunning, professional resumes effortlessly with this **Canva-inspired Resume Builder**. Designed with a modern UI and real-time preview, this tool helps users create job-ready resumes and export them as high-quality PDFs instantly.

---

## 🔥 Overview

This project is a **front-end web application** that enables users to:

* Create resumes interactively
* Preview changes in real time
* Customize templates and styles
* Export resumes in PDF format

It replicates the experience of platforms like **Canva, Resume.io, and Novoresume** using pure front-end technologies.

---

## 🚀 Core Features

### 🎨 Design & Customization

* Multiple modern resume templates
* Dynamic theme and color switching
* Profile photo integration

### ⚡ Interactive Editing

* Live preview editor
* Drag-and-drop section arrangement
* Instant content updates

### 💾 Data Management

* Save resume locally
* Load previously saved resumes
* No database required

### 📄 Export

* One-click PDF download
* ATS-friendly layout support
* High-resolution export
